Detailed Description

Builder class for creating Blade objects.

Each of the methods for setting options returns a reference to this BeamBuilder object to allow chaining calls together in a single statement.

◆ SetQuadratureStyle()

BeamBuilder & kynema_fmb::interfaces::components::BeamBuilder::SetQuadratureStyle ( BeamInput::QuadratureStyle  style)

Sets the Quadrature style to use for this beam.

Segmented applies the quadrature rule between each of the specified sectional properties. WholeBeam has just one quadrature applied to the entire beam.

Combining GLL quadrature with the Segmented style wil preserve the user's specified sectional properties at their exact locations. This is expected to give better performance for complex property distributions.

WholeBeam will likely be the most efficient for simple property distributions.

Returns
A reference to this BeamBuilder
